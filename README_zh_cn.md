ReCoLit是一个面向团队协作的一站式文献管理工具，致力于打通团队文献共享、阅读与知识库建设全流程。

它聚焦团队协作场景，提供轻量化、高效率的文献共享、精准检索与团队知识库构建能力，打破内部文献传递壁垒，实现文献资源集中化、规范化管理，全面提升团队文献管理效率。

产品创新性融合专业网盘级存储传输与学术文献阅读标注能力：
- 支持团队文献安全存储、快速共享与精细化权限管控，成员可高效获取所需资源
- 内置文献批注、高亮、笔记等核心阅读功能，无需多工具切换
- 一站式完成文献获取 → 阅读 → 标注 → 管理 → 协作全流程操作
显著降低团队文献管理的时间成本与学习成本

## 快速开始

在一切开始前，你需要克隆这个项目：
```bash
git clone xxxxxx
```

### 前端
在运行前端之前，请确保你有以下的软件：
- `nodejs`

进入前端的根目录然后运行`npm install`来安装依赖：
```bash
cd frontend
npm install
```

在上面的工作完成后，就可以运行前端项目：
```
npm run dev
```
你现在可以在 [http:localhost:5173](http:localhost:5173) 上访问到前端页面
